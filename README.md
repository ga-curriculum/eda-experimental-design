## About
An introduction to the framework of experiments, hypothesis testing, sources of bias in experiments, and how they relate specifically to A/B testing.


## Learning Objectives

*You will be able to:*

- Explain the main reasons to conduct experiments.
- Understand the framework of conducting an experiment.
- Define key components of an experiment including the outcome, sources of variation, and the necessary data.
- Explain how experimental design relates to A/B tests.
- Define the key steps and terms in hypothesis testing.
